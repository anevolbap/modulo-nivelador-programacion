Clase 5 - Ejercicios

Simulación, algoritmos iterativos y mini-proyecto

Contexto

Última guía. Construís dos generalizaciones (bootstrap_estimador, iterar) y un algoritmo iterativo (biseccion). Cerramos con un mini-proyecto que junta toda la caja de herramientas sobre sysarmy.

Reusás: bootstrap_media (clase 2), eda_minimo y graficar_distribucion (clase 4), cargar_y_describir (clase 3).

sueldos_url <- "https://raw.githubusercontent.com/openqube/openqube-sueldos/master/data/csv/argentina/2021.02.csv"

Ejercicio 1: función iterar

  1. Definí iterar(f, x0, n) que aplique f al valor n veces, partiendo de x0. Devolvé el valor final.

  2. Probala con f <- function(x) (x + 2/x) / 2, x0 = 1, n = 10. ¿A qué número converge? Pista: es la iteración Babilónica para sqrt(2).

  3. Modificá iterar para que devuelva la trayectoria completa (vector de longitud n + 1). Graficala con plot(..., type = "b").

Ejercicio 2: Monte Carlo con monedas

  1. Estimá P(al menos 6 caras de 10 monedas equilibradas) con replicate + sample(c(0, 1), 10, replace = TRUE). Usá 10000 réplicas.

  2. Compará con 1 - pbinom(5, 10, 0.5).

  3. Repetí para n = 100, 1000, 10000 monedas (proporción >= 0.6). Graficá la estimación contra n. ¿Converge?

Ejercicio 3: bootstrap_estimador generalizado

  1. Definí bootstrap_estimador(x, estimador = mean, n_rep = 1000) que reemplace tu bootstrap_media (clase 2). El argumento estimador es una función.

  2. Probala con mean, median y sd sobre airquality$Wind. Calculá IC del 95% en cada caso.

  3. ¿Cómo se relaciona con bootstrap_media? Reescribí bootstrap_media como un wrapper de bootstrap_estimador con estimador = mean por defecto.

Ejercicio 4: paradoja del cumpleaños

  1. Definí prob_coincidencia(n, n_sim = 10000) que estime la probabilidad de coincidencia en un grupo de n personas. Pista: any(duplicated(sample(1:365, n, replace = TRUE))).

  2. Aplicala a n = c(5, 10, 15, 20, 23, 30, 50) y graficá los resultados con plot(n, p, type = "b").

  3. Encontrá el n mínimo para P > 0.5 recorriendo de a 1 con un while. ¿Coincide con la respuesta clásica (23)?

Ejercicio 5: bisección

  1. Implementá biseccion(f, a, b, tol = 1e-6) con while. Usá stopifnot(f(a) * f(b) < 0) como precondición.

  2. Encontrá la raíz de f(x) = cos(x) - x en [0, 1]. Compará con uniroot(f, c(0, 1))$root.

  3. Modificá biseccion para que devuelva una lista con raiz, iteraciones y tolerancia_final. Comparala con uniroot(...)$iter.

Ejercicio 6: bootstrap sobre sysarmy

  1. Cargá sysarmy y quedate con la columna del salario neto (la del 8b/8c de la guía 4). Aplicá as.numeric si hace falta y filtrá NA.

  2. Aplicá bootstrap_estimador(salario_neto, estimador = median, n_rep = 1000). Calculá el IC del 95%.

  3. Compará la mediana puntual con el IC. Discutí (en una línea) por qué el IC del bootstrap es informativo cuando la distribución es asimétrica.

Ejercicio 7: regresión sobre sysarmy

  1. Identificá la columna de “años de experiencia” (algo como Años.de.experiencia). Convertí a numérica si hace falta. Filtrá NA y outliers extremos (por ejemplo, salarios sobre el percentil 99).

  2. Ajustá lm(salario_neto ~ años_experiencia). Mostrá summary(modelo)$coefficients y confint(modelo).

  3. Hacé bootstrap del coeficiente de “años de experiencia” con replicate. En cada réplica: sample de filas con reemplazo, ajustar lm, extraer el coeficiente. Calculá IC del 95% y compará con confint.

Ejercicio 8: graficar_bootstrap

  1. Definí graficar_bootstrap(boot, etiqueta = "") que: dibuje el histograma del vector boot, marque la media con una vertical roja, y los cuantiles 2.5% y 97.5% con dos verticales punteadas.

  2. Aplicala al bootstrap de la mediana de salario (ejercicio 6).

  3. Aplicala al bootstrap del coeficiente de regresión (ejercicio 7).

Ejercicio 9: mini-pipeline integrador

  1. Definí analizar_columna(df, columna, estimador = mean) que: aplique eda_minimo (clase 4) sobre el df, ejecute bootstrap_estimador sobre la columna indicada, y llame a graficar_bootstrap. Devolvé una lista con los resultados.

  2. Aplicala a la columna de salario neto de sysarmy con estimador = median.

  3. Aplicala a una columna numérica de airquality con estimador = mean. ¿Qué te dice cada salida?

Ejercicio 10: control de versiones

  1. En la carpeta donde tenés guardadas todas las funciones del curso, hacé git init. Mirá git status.

  2. Hacé tres commits con mensajes que expliquen el porqué:

    • uno con etiquetar_auto, dif_promedio, resumen_min (clase 1),
    • otro con clasificar_temp, bootstrap_media, clasificar_presion (clase 2),
    • un último con medias_por_grupo, cargar_y_describir, graficar_distribucion, eda_minimo, bootstrap_estimador, biseccion, analizar_columna.

  3. Mirá git log --oneline. ¿El historial cuenta una historia clara de lo que aprendiste? Si no, ¿qué cambiarías?
