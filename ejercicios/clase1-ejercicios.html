Clase 1 - Ejercicios

Tipos de datos, vectores y primeras funciones

Contexto

Vamos a usar mtcars, un dataset de 32 autos con 11 variables (consumo, cilindros, potencia, peso, etc.). Viene con R, no requiere instalación.

head(mtcars)
?mtcars

A lo largo de esta guía construimos una pequeña caja de herramientas que vamos a reusar en las clases siguientes: etiquetar_auto, dif_promedio, dentro_rango, resumen_min. Los ejercicios marcados como caja negra usan funciones del lenguaje sin pedir que las implementen.

Ejercicio 1: variables y tipos atómicos

  1. Asigná cuatro variables que describan el primer auto de mtcars: nombre_auto (string con rownames(mtcars)[1]), consumo_mpg (double, mtcars$mpg[1]), cilindros (integer, as.integer(mtcars$cyl[1])) y transmision_manual (logical, mtcars$am[1] == 1). Verificá class() de cada una.

  2. Verificá también typeof(). ¿En qué se diferencian class y typeof para consumo_mpg?

  3. ¿Qué devuelve is.numeric(cilindros)? ¿Y is.numeric(transmision_manual)? Justificá.

Ejercicio 2: coerción de tipos

  1. Predecí (sin correr) qué devuelve c(TRUE, 1L, 1.5). Después corrélo y compará con tu predicción.

  2. Predecí qué devuelve c(TRUE, 1, "a"). Justificá usando la regla de coerción.

  3. ¿Qué pasa al evaluar consumo_mpg + transmision_manual? Identificá qué tipo predomina.

Ejercicio 3: vector de consumos

  1. Asigná consumos <- mtcars$mpg. Verificá length(consumos) y class(consumos).

  2. Calculá mean, median, min y max de consumos. Compará con la salida de summary(consumos).

  3. Quedate con los consumos por encima del promedio usando indexado lógico. ¿Cuántos son?

Ejercicio 4: indexar mtcars

  1. Mostrá la primera fila completa de mtcars, la columna hp entera y la celda fila 5 columna mpg.

  2. Encontrá el auto con mayor mpg con which.max. Mostrá su nombre con rownames(mtcars)[which.max(mtcars$mpg)].

  3. Mostrá las primeras tres columnas (mpg, cyl, disp) de los primeros cinco autos. Verificá la estructura del resultado.

Ejercicio 5: filtros con condiciones múltiples

  1. Filtrá los autos con mpg > 25. ¿Cuántos son?

  2. Filtrá los autos con mpg > 25 Y cyl == 4. Pista: operador &.

  3. Filtrá los autos con cyl == 4 o cyl == 8. Pista: | o %in%.

Ejercicio 6: función etiquetar_auto

  1. Definí etiquetar_auto(modelo, mpg) que devuelva el string "<modelo>: <mpg> mpg". Pista: paste0. Probala con "Mazda RX4" y 21.

  2. Probala con valores extraídos de mtcars: etiquetar_auto(rownames(mtcars)[1], mtcars$mpg[1]).

  3. Agregale un valor por defecto mpg = NA y observá la salida cuando se omite el segundo argumento.

Ejercicio 7: función dif_promedio

  1. Definí dif_promedio(x) que devuelva x - mean(x). Aplicala a mtcars$mpg. Calculá la suma de las diferencias. Justificá el resultado teóricamente.

  2. ¿Cuántos autos están por encima del promedio? Pista: sum(dif_promedio(x) > 0).

  3. ¿Cuál es el auto que más se aleja del promedio en valor absoluto? Pista: which.max(abs(...)).

Ejercicio 8: función dentro_rango

  1. Definí dentro_rango(x, a, b) que devuelva TRUE si a <= x <= b, FALSE en otro caso. La función debe ser vectorizada: si x es un vector, devuelve un vector de booleanos. Pista: &.

  2. Probala con dentro_rango(c(1, 5, 10, 15, 20), 5, 15).

  3. Aplicala a mtcars$mpg con rango c(20, 25). ¿Cuántos autos caen dentro del rango?

Ejercicio 9: conversión de unidades

  1. Definí mpg_a_km_l(x) que convierta de mpg (millas por galón) a km/L. Factor: 1 mpg ≈ 0.4251 km/L.

  2. Aplicala a mtcars$mpg para obtener un vector de consumos en km/L.

  3. Definí km_l_a_mpg(x) (función inversa). Verificá que km_l_a_mpg(mpg_a_km_l(20)) devuelve aproximadamente 20.

Ejercicio 10: función resumen_min

  1. Definí resumen_min(x) que reciba un vector numérico y devuelva una lista con media, mediana, min, max.

  2. Aplicala a mtcars$mpg. Verificá class() del resultado.

  3. Aplicala a mtcars$hp. Compará el rango con el de mpg.

Ejercicio 11: lista que describe un auto

  1. Creá una lista auto_1 con: modelo (string), mpg (double), cilindros (integer), caja ("manual" o "automatica"). Usá los datos del primer auto de mtcars.

  2. Accedé a cada campo con $, con [[ ]] y con [ ]. Mostrá class() de cada acceso.

  3. Cambiá el valor de mpg reasignándolo (auto_1$mpg <- ...). Verificá.

Ejercicio 12: comparar dos autos

  1. Definí comparar_autos(auto_a, auto_b) que reciba dos listas con campos mpg y cilindros y devuelva una lista con dif_mpg y dif_cilindros.

  2. Probala con dos autos cualesquiera de mtcars.

  3. Si llamás comparar_autos(list(mpg = 20), list(consumo = 25)), ¿qué pasa? Identificá el problema.

Ejercicio 13: ordenar por consumo

  1. Ordená mtcars por mpg descendente. Pista: order(-mtcars$mpg) o order(..., decreasing = TRUE).

  2. Mostrá los nombres de los tres autos con mayor mpg.

  3. Mostrá los tres con menor mpg.

Ejercicio 14: aplicar resumen_min a varias columnas

  1. Aplicá resumen_min por separado a las columnas mpg, hp y wt de mtcars. Guardá los resultados en una lista nombrada.

  2. Hacé lo mismo en una sola línea con lapply. Pista: lapply(mtcars[, c("mpg", "hp", "wt")], resumen_min).

  3. ¿Qué tipo devuelve lapply y qué devuelve sapply? Compará los resultados.

Ejercicio 15: tu propio data.frame

  1. Creá un data.frame con tres pacientes y cuatro columnas: id (integer), edad (double), peso_kg (double), fuma (logical).

  2. Aplicá resumen_min a la columna edad.

  3. Aplicá summary() al data.frame completo. Compará la salida con la de resumen_min.

Ejercicio 16: quantile (caja negra)

  1. Aplicá quantile(mtcars$mpg) (sin argumentos extra). Identificá el percentil 25, la mediana y el percentil 75.

  2. Aplicá quantile(mtcars$mpg, probs = c(0.1, 0.5, 0.9)). ¿Qué representa cada valor?

  3. Definí rango_intercuartilico(x) usando quantile. Aplicalo a mtcars$mpg y compará con IQR(mtcars$mpg).

Ejercicio 17: correlación con cor (caja negra)

  1. Calculá cor(mtcars$mpg, mtcars$hp). Interpretá el signo y la magnitud (cor toma valores entre -1 y 1).

  2. Calculá la matriz de correlaciones entre mpg, hp, wt y cyl con cor(mtcars[, c("mpg", "hp", "wt", "cyl")]).

  3. Identificá el par de variables más correlacionado y el menos correlacionado.

Ejercicio 18: leer la documentación

  1. Abrí ?sapply. Identificá los argumentos X, FUN y simplify.

  2. Probá sapply(1:5, function(i) i^2). ¿Devuelve un vector o una lista?

  3. Probá lapply(1:5, function(i) i^2). Compará con la salida de sapply y explicá cuándo conviene cada uno.

Ejercicio 19: la caja de herramientas sobre datos sintéticos

  1. Generá un vector x con rnorm(100, mean = 50, sd = 10) (100 valores normales). rnorm es una caja negra: ver ?rnorm.

  2. Aplicale resumen_min, dif_promedio y dentro_rango con a = 40, b = 60.

  3. ¿Aproximadamente cuántos valores caen en [40, 60]? Compará con la regla 68-95-99.7 de la distribución normal (esperado: ~68%).

Ejercicio 20: buenos nombres

  1. Tomá este snippet:

    f <- function(x, y) x + y * 0.6213
a <- 100
print(f(0, a))

    Pista: el factor 0.6213 convierte kilómetros a millas.

  2. Reescribilo cambiando los nombres para que la intención sea clara sin necesidad de comentarios. Pensá nombres descriptivos para f, a, x, y.

  3. Justificá tu elección en una línea.
